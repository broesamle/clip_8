clip_8 | Tutorial

Welcome to the `clip_8` language tutorial. Clip_8 is a programming language. Unlike most other programming languages, the instructions are icons. You will draw the program code in SVG.

The tutorial assumes you have some experience with vector graphics editors.

If you get stuck, please read the Survival Guide! as the tutorials and clip_8 are in an experimental state.

Happy drawing!
